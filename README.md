# VirtualJTC
fork of jtcemu

VirtualJTC is a fork of jtcemu http://www.jens-mueller.org/jtcemu/ 
and is a Emulator of the JU+TE Computer http://hc-ddr.hucki.net/wiki/doku.php/tiny or JTC for short.

This computer was based on an east german Zilog Z8 clone, the UB 883, which contains a TinyBasic in ROM.

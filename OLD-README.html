JTCEMU

1. Allgemeines

4. Installation

  1. Installieren Sie die Java Runtime Environment (JRE) der Java Standard Edition (Java SE) Version 6 oder höher. Überprüfen Sie die installierte Java-Version durch den Kommandozeilenaufruf: java -version Für Java 6 muss die Versionsnummer 1.6.x erscheinen.
  2. Laden Sie die Datei jtcemu-1.1.jar von http://www.jens-mueller.org/jtcemu herunter. Die Datei darf dabei nicht als ZIP-Datei entpackt werden!
  3. Kopieren Sie die Datei jtcemu-1.1.jar in ein beliebiges Verzeichnis auf die Festplatte ihres Computers, z.B. unter Linux/Unix nach /tmp oder unter Windows nach C:\
  4. Starten Sie JTCEMU
    • unter Linux/Unix mit: java -jar /tmp/jtcemu-1.1.jar
    • unter Windows in der DOS-Box mit: javaw.exe -jar C:\jtcemu-1.1.jar
  5. Legen Sie auf dem Desktop eine Verknüpfung zu einem Programm an, und tragen dort die in der Console/DOS-Box erfolgreich verwendete Kommandozeile ein.


5. Compilieren

Möchten Sie den Quelltext compilieren, sind folgende Schritte notwendig:
  1. Stellen Sie sicher, dass das Java Development Kit (JDK) der Java Standard Edition (Java SE) Version 6 oder höher installiert ist.
  2. Wechseln Sie in das cmd-Verzeichnis des JTCEMU-Quelltextes
  3. Compilieren Sie
    • unter Linux/Unix mit: ./compile
    • unter Windows in der DOS-Box mit: compile.cmd
Alternativ können Sie auch mit ant compilieren. Die dazu notwendige Datei build.xml finden Sie im Wurzelverzeichnis des Quelltextes.

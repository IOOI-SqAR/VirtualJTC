Hilfeseiten drucken

Sie können die einzelnen Hilfeseiten über das Menü Datei ausdrucken. Dabei erscheinen die Seiten auf dem Drucker so, wie sie auch im Fenster angezeigt werden. Ist die Fensterbreite größer als auf das Papier passt, wird die Seite entsprechend verkleinert. Sie können das Druckbild somit beeinflussen, indem Sie das Hilfefenster entsprechend breiter oder schmaler ziehen.

Eine andere Möglichkeit besteht darin, die angezeigte Hilfeseite mit oder ohne Links in die Zwischenablage zu kopieren (siehe Menü Datei) und dann in eine Textverarbeitung ihrer Wahl einzufügen, dort ggf. noch etwas zu formatieren und dann zu drucken. Das Textverarbeitungsprogramm muss dazu in der Lage sein, HTML-formatierten Text einfügen zu können. Das sollten aber alle aktuell üblichen Textverarbeitungen können.

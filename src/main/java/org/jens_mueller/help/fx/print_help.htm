Hilfeseiten drucken

JTCemuFX bietet selbst keine Möglichkeit, Hilfeseiten zu drucken. Sie können aber jede Hilfeseite mit oder ohne Links in die Zwischenablage kopieren (siehe Menü Datei) und dann in eine Textverarbeitung ihrer Wahl einfügen, dort ggf. noch etwas formatieren und dann drucken.

Das Textverarbeitungsprogramm muss dazu in der Lage sein, HTML-formatierten Text einfügen zu können. Das sollten aber alle aktuell üblichen Textverarbeitungen können.

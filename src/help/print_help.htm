Hilfeseiten drucken

Sie können die einzelnen Hilfeseiten über das Menü Datei ausdrucken. Dabei erscheinen die Seiten auf dem Drucker so, wie sie auch im Fenster angezeigt werden. Ist die Fensterbreite größer als auf das Papier passt, wird die Seite entsprechend verkleinert.

Sie können das Druckbild somit beeinflussen, indem Sie das Hilfefenster entsprechend breiter oder schmaler ziehen.
